 Açaiteria

Programa em Python que simula uma açaiteria com um menu interativo no terminal.
	•	Exibe produtos mais populares e opções disponíveis
	•	Permite montar e personalizar pedidos
	•	Mostra promoções, preços e formas de pagamento
	•	Informa opções de entrega (delivery)
	•	Oferece suporte ao cliente e coleta feedback

O sistema funciona por meio de um menu com escolhas numéricas, sendo ideal para praticar lógica de programação e interação com o usuário
